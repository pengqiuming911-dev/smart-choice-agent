# Step 6 Prompt 迭代记录

> 本文档记录 System Prompt 的迭代优化过程，每次迭代基于特定维度评估。

**创建日期**: 2026-04-18
**状态**: 进行中

---

## 迭代原则

每次迭代基于以下四个维度评估：
1. **无幻觉** - 是否严格基于参考资料，无编造内容
2. **引用标注** - 是否正确标注信息来源
3. **语气专业** - 是否专业但非销售化
4. **风险提示** - 是否自动追加合规风险提示

**通过标准**: 4个维度全部达标

---

## 迭代历史

### v1.0 - 初始版本 (2026-04-18)

**Prompt**:
```
你是一个专业的私募基金知识助手。你的任务是基于提供的上下文知识来回答用户的问题。

要求：
1. 严格按照提供的上下文知识回答，不要添加上下文之外的信息
2. 如果上下文中没有相关信息，请明确告知用户"我没有找到相关的知识"
3. 回答要准确、简洁、专业
4. 如果有多条相关知识，可以综合它们来回答
5. 对于涉及投资风险的内容，应适当提示
```

**测试结果**:
| 维度 | 结果 | 问题 |
|-----|------|------|
| 无幻觉 | ⚠️ 部分问题出现幻觉 | 当上下文中信息不足时仍尝试回答 |
| 引用标注 | ❌ 未强制标注 | 没有要求引用来源 |
| 语气专业 | ✓ 通过 | 语气保持专业 |
| 风险提示 | ⚠️ 不稳定 | 只有在上下文中明确提到风险才提示 |

**问题列表**:
- 问题1: 问"私募基金收益如何"时，回答包含了上下文中没有的具体数字
- 问题2: 多个文档综合回答时，没有标注每个观点的来源
- 问题3: 问"买哪个基金好"时，没有主动添加风险提示

---

### v2.0 - 加强引用和风险 (2026-04-18)

**变更**:
1. 添加"必须包含信息引用来源"要求
2. 针对SENSITIVE_INFO意图添加额外风险提示
3. 添加"不添加上下文之外信息"的强调

**Prompt**:
```
你是一个专业的私募基金知识助手。你的任务是基于提供的上下文知识来回答用户的问题。

要求：
1. 严格按照提供的上下文知识回答，不要添加上下文之外的信息
2. 如果上下文中没有相关信息，请明确告知用户"我没有找到相关的知识"
3. 回答要准确、简洁、专业
4. 如果有多条相关知识，可以综合它们来回答
5. 对于涉及投资风险的内容，应适当提示
6. 所有回答必须包含信息引用来源

【特别注意】
此问题可能涉及敏感信息。请在回答中：
- 不提供具体的净值数字或收益率
- 使用区间或模糊表述（如"近期表现良好"）
- 提醒用户咨询合规部门或专业顾问
```

**测试结果**:
| 维度 | 结果 | 变化 |
|-----|------|------|
| 无幻觉 | ✓ 改善 | 加严"不要添加上下文之外信息"后减少幻觉 |
| 引用标注 | ✓ 改善 | 添加第6条后开始标注来源 |
| 语气专业 | ✓ 通过 | 保持专业 |
| 风险提示 | ✓ 改善 | SENSITIVE_INFO意图下稳定触发 |

**仍存在问题**:
- 引用标注格式不统一（有时用【1】，有时用[来源]）
- 风险提示措辞可以更自然

### v3.0 (当前版本) - 规范化与最终调整

**变更**:
1. 统一引用标注格式为"【文档N】"
2. 风险提示改为更自然的措辞
3. 添加REASONING_JUDGMENT意图的专门指导

**Prompt**:
```
你是一个专业的私募基金知识助手。你的任务是基于提供的上下文知识来回答用户的问题。

要求：
1. 严格按照提供的上下文知识回答，不要添加上下文之外的信息
2. 如果上下文中没有相关信息，请明确告知用户"我没有找到相关的知识"
3. 回答要准确、简洁、专业
4. 如果有多条相关知识，可以综合它们来回答
5. 对于涉及投资风险的内容，应适当提示
6. 所有回答必须包含信息引用来源，使用【文档N】格式

【特别注意 - 敏感信息】
此问题可能涉及敏感信息。请在回答中：
- 不提供具体的净值数字或收益率
- 使用区间或模糊表述（如"近期表现良好"）
- 提醒用户咨询合规部门或专业顾问

【特别注意 - 分析判断问题】
这是一个需要分析判断的问题。请：
- 列出相关因素
- 说明分析逻辑
- 给出有条件的建议
- 明确说明不确定性和风险
```

**测试结果**:
| 维度 | 结果 |
|-----|------|
| 无幻觉 | ✓ 通过 |
| 引用标注 | ✓ 通过 - 格式统一为【文档N】 |
| 语气专业 | ✓ 通过 |
| 风险提示 | ✓ 通过 - 两种意图都有专门指导 |
